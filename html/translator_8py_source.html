2 """
3 DESCRIPTION
4 This web service written in Python allows you to translate a string into the language of your choice.
5 To do this, we ask you to fill out a form containing 2 inputs:
6  - An input to write a string.
7  - An input to select the language of translation.
8 Once you have written your string, selected the language of translation and clicked on translate, this
9 web service will interoperate with the Google translate API by sending as a parameter in the URL the
10 string you have written and the language of translation. If the request has been taken by the Google
11 API, it sends back to the web service a response containing the translated text in a JSON format.
12 The web service retrieves this information and records: the date and time of the translation, the
13 string written, the language of translation and the string translated into a CSV file. This in order
14 to give you the translation of your input but also to provide you an history of all your translations.
15 As a result, the web service redirects you to the form. The latter re-displays the translation form as
16 well as the information contained in the CSV file which will give you the history of your translations
17 and offer you the possibility to perform several successive translations.
18 """
