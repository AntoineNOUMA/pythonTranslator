19 ## Description
20 
21 Ce web service écrit en Python vous permet de traduire une chaîne de caractère saisi au clavier dans la langue de votre choix.
22 
23 Pour ce faire, nous vous demandons de remplir un formulaire contenant 2 entrées :
24 
25  - Une entrée pour saisir une chaîne de caractère.
26  - Une entrée pour selectionner la langue de traduction.
27 
28 Une fois que vous avez saisie votre chaîne de caractère, que vous avez selectionné la langue de traduction et que vous avez cliqué sur traduire, ce web service va allé intéroger l'API Google translate en lui envoyant en paramètre dans une URL la chaîne de caractère que vous avez saisie ainsi que la langue de traduction. Si la requête à bien été prise en compte par l'API de Google, elle renvoie au web service une réponse contenant le texte traduit en format JSON.
29 
30 Le web service récupère donc cette information et enregistre : la date de la traduction, l'heure de la traduction, la chaîne de caractère saisie, la langue de traduction ainsi que la chaîne de caractère traduite dans un fichier CSV. Ceci dans le but de vous restituer la traduction de votre saisie mais aussi de vous fournir un historique de toute vos traductions.
31 
32 Par suite, le web service vous redirige vers le formulaire. Ce dernier vous ré-affiche le formulaire de traduction ainsi que les informations contenues dans le fichier CSV ce qui en somme vous donnera l'historique de vos traductions et vous offrira la possibilité d'effectuer plusieurs traductions successive.

40 ### Procédure d'installation des composants de l'application
41 
42 Dans ce qui suit, vous allez devoir faire des installations via le terminal. Si vous êtes relié à un réseaux d'entreprise, il se peut que certaines commandes notamment pour réaliser un téléchargement, ne fonctionne pas. C'est pourquoi je vous conseil de débrancher le câble ethernet de votre PC est de vous mettre en partage de connexion via votre mobile durant la phase d'installation.
43 
44 Pour faire fonctionner ce web service écris en Python, vous avez besoin premièrement de **Python**!
45 
46 Il existe 2 versions de Python, Python 2 et Python 3. Le code de ce web service fût codé et testé sous Python 2, il est donc garantie qu'il fonctionne sous cette version.
47 
48 Pour savoir si Python est déjà installé sur votre machine, il suffit d'ouvrir un termminal et d'exécuter la commande `python`. Si le terminal vous affiche quelques chose de ce style :
49 
50 _Python 2.7.14 (v2.7.14:84471935ed, Sep 16 2017, 20:19:30) [MSC v.1500 32 bit (Intel)] on win32_
51 
52 _Type "help", "copyright", "credits" or "license" for more information._
53 
54 c'est que Python est déjà installé sur votre machine!
55 
56 Si ce n'est pas la cas ou si vous souhaitez installer la dernière version de Python 2 ou Python 3, cliquez sur le lien suivant : **https://www.python.org/downloads/**.
57 
58 Une fois Python installé, vous avez besoin d'installer plusieurs paquets Python dont la liste exhaustive avec une descriptinon succinte de chacun d'eux se trouve dans les deux tableaux ci-dessous. Le premier regroupe les paquets Python natif c'est à dire ceux que l'on peux importer en ayant juste installé Python. Le second regroupe les paquets Python non natif, c'est à dire qu'il faut les installer via pip avant de les importer.
59 
60 Pour installer ces paquets non natif, vous utiliserez la commande `pip install` nomDuPaquet dans le terminal. Pour que cette dernière fonctionnne, vous devez préalablement installer le gestionnaire de paquets **pip**!
61 
62 Pip est déjà installé si vous travaillez dans un environnement virtuel créé par virtualenv ou pyvenv ou si vous utilisez des binaires Python à partir des versions 2.7.9 et 3.4 téléchargés depuis python.org, mais vous devrez mettre à jour pip!
63 
64 Pour installer pip, il suffit de se rendre sur la page de téléchargement suivante : **https://pip.pypa.io/en/stable/installing/**. Une fois le téléchargement éffectuer, exécuter la commande suivante dans votre terminal : `python get-pip.py`

88 ### Paramètre en entrée
89 
90 * Une chaîne de caractères saisie par l'utilisateur sur "/formulaire".
91 * Une langue de traduction choisie dans la liste déroulante par l'utilisateur sur "/formulaire".

95 ### Sortie possible
96 
97 * Si tous s'exécute correctement
98  * Le formulaire pour effectuer une nouvelle traduction.
99  * Un tableau HTML composé :
100  * de la date de la traduction.
101  * de l'heure de la traduction.
102  * du texte original.
103  * de la langue de traduction.
104  * du texte traduit.
105  * de l'historique des traductions.
106 * Si la réponse du serveur de Google est differente de 200 :
107  * Un message d'erreur.
108  * Le formulaire.
109 * Si vous cliquez sur "traduire" sans avoir préalablement saisie une chaine de caractère :
110  * Un message d'erreur.
111  * Le formulaire.

115 ### Gestion des erreurs
116 
117 | Erreur | Gestion de l'erreur |
118 |--------|---------------------|
119 | Envoie d'une chaîne de caractères vide dans "/formulaire" | On renvoie l'utilisateur vers "/formulaire" avec le message suivant : _Erreur : veuillez saisir quelque chose à traduire._|
120 | Réponse du serveur de Google differente de 200 | On renvoie à l'utilisateur un message d'erreur : _Erreur : connexion à l'API Google translate impossible._|
121 | Time out | On renvoie à l'utilisateur un message d'erreur : _Erreur : à faire_|
